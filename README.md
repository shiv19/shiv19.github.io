## My Personal Blog

This is a Jekyll-based personal blog hosted on GitHub Pages.
Here's the tech stack:

#### Core Technologies

- Jekyll 4.3 - Static site generator (Ruby-based)
- Ruby - Backend/build environment
- Liquid - Templating language
- Markdown/Kramdown - Content authoring

#### Frontend

- Vanilla JavaScript - No major framework for main site
- SCSS/SASS - CSS preprocessing with modular partials
- CSS Variables - Modern theming (dark/light mode)
- Google Fonts - Inter & JetBrains Mono

#### Jekyll Plugins

- jekyll-sitemap, jekyll-feed, jekyll-paginate, jemoji

Features & Integrations
- PWA Support - Progressive Web App with service workers (@pwabuilder/pwaupdate)
- Custom dark/light mode toggle
- Disqus - Comments
- Google Analytics
- Rouge - Syntax highlighting

#### Embedded Apps

The blog includes several embedded applications:
- Stridemate - Angular-based app
- Space Survival - Game with bundled JS
- Battleship - Python/Skulptor game
- Dragster - Bootstrap-based app

#### Build & Deployment

- Jekyll build system with Sass compilation
- GitHub Pages deployment (github.io domain)

This is a modern, performance-focused static blog with custom JavaScript enhancements and no heavy framework dependencies for the core functionality.
